The CSS Box Model

The four parts of the CSS box model

The CSS box model breaks down an HTML element into four sections that can be styled. Let's take a closer look at each section.

  1. Content
  2. Padding
  3. Border
  4. Margin

Every HTML element has a content box, which holds the actual content like text, images, or other HTML elements. The content box automatically adjusts to fit whatever is inside it, but you can customize its size using the CSS width and height properties.

Around the content is the padding, border, and margin. The size of each of these can be set using CSS.
